Allow new users to register /users POST Adds new user JSON data to user JSON list JSON required
{
Username: { type: String, required: true },
Password: { type: String, required: true },
Email: { type: String, required: true },
Birthday: Date,
FavoriteMovies: [{ type: mongoose.Schema.Types.ObjectId, ref: "Movie" }],
}
Allow users to update their user info /users/:Username PUT A text message indicating whether the username was successfully updated JSON required with updated data
{
Username: String,
(required) Password: String,
(required) Email: String,
(required) Birthday: Date
}
Allow users to add a movie to their list of favorites /users/:Username/movies/:MovieID POST A text message confirming the movie in URL param was added successfully
Allow users to remove a movie from their list of favorites /users/:Username/movies/:MovieID DELETE A text message confirming the movie in URL param was removed successfully
Allow existing users to deregister /users/:Username DELETE A text message confirming the user was deregistered
