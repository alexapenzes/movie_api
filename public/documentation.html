Feature URL Request Response
Return a list of all movies to the user /movies GET A JSON object holding data over all movies
Return data about a single movie by title to the user /movies/:title GET A JSON object for individual movie passed in URL param
Return data about a genre (description) by name movies/genre/:genreName GET A JSON object for individual genre passed in URL param
Return data about a director by name movies/directors/:directorName GET A JSON object for director passed in URL param
Allow new users to register /users POST
w/ JSON body param {"name": "", "age":""}		 Adds new user JSON data to user JSON list
Allow users to update their user info (username) /users/:id PUT A text message indicating whether the username was successfully updated
Allow users to add a movie to their list of favorites (showing only a text that a movie has been added) /users/:id/:movieTitle POST A text message confirming the movie in URL param was added successfully
Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed) /users/:id/:movieTitle DELETE A text message confirming the movie in URL param was removed successfully
Allow existing users to deregister (showing only a text that a user email has been removed) /users/:id DELETE A text message confirming the user was deregistered
